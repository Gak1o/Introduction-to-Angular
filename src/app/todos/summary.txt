What is a Component?

A component controls a part of the UI (view).
Every Angular app is made of components.

A component has:

HTML template ‚Üí what the user sees

TypeScript class ‚Üí logic & data

CSS (optional) ‚Üí styling

Component Structure
import { Component } from '@angular/core';

@Component({
  selector: 'app-hello',
  template: `<h1>Hello {{ name }}</h1>`,
  styleUrls: ['./hello.component.css']
})
export class HelloComponent {
  name = 'Angular';
}

Key Parts

selector ‚Üí used in HTML: <app-hello></app-hello>

template / templateUrl ‚Üí UI

class ‚Üí logic

2Ô∏è‚É£ Angular Services
What is a Service?

A service handles:

Business logic

Data fetching (API calls)

Shared data between components

üëâ Keeps components clean & reusable

Creating a Service
ng generate service services/todos

Service Example
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class TodosService {
  getTodos() {
    return ['Learn Angular', 'Build App', 'Deploy'];
  }
}

Using a Service in a Component
import { Component } from '@angular/core';
import { TodosService } from '../services/todos.service';

@Component({
  selector: 'app-todos',
  template: `
    <ul>
      <li *ngFor="let todo of todos">{{ todo }}</li>
    </ul>
  `
})
export class TodosComponent {
  todos: string[];

  constructor(private todosService: TodosService) {
    this.todos = this.todosService.getTodos();
  }
}

What is Data Binding?

Data binding connects the component class to the HTML template.

Angular has 4 types üëá

1Ô∏è‚É£ Interpolation (One-way)

Component ‚Üí View

2Ô∏è‚É£ Property Binding

Component ‚Üí HTML property

3Ô∏è‚É£ Event Binding

HTML ‚Üí Component

  alert('Hello!');
}

4Ô∏è‚É£ Two-Way Binding

Component ‚Üî View (requires FormsModule)
